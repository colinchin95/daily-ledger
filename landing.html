Install in 10 seconds

Centsei is a web app — no app store needed. Add it to your home screen and it behaves like a native app.

iPhone / iPad
  1. Open Centsei in Safari
  2. Tap the Share button
  3. Tap Add to Home Screen
🤖 Android
  1. Open Centsei in Chrome
  2. Tap the ⋮ menu
  3. Tap Install app
💻 Desktop
  1. Open Centsei in Chrome / Edge
  2. Click the Install icon in the address bar
  3. Launch it like any app
